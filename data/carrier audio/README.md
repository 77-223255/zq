# 说明
## 📂 数据集目录结构

| 文件夹名称        | 采样率   | 处理阶段                | 描述                                                                 |
|-------------------|----------|-------------------------|----------------------------------------------------------------------|
| `8/`              | 8 kHz    | 原始音频                | 未经处理的原始音频文件                                               |
| `16/`             | 16 kHz    | 重采样处理              | 对 `8/` 目录下音频进行重采样得到                 |
| `16_normalized/`  | 16 kHz    | 归一化处理              | 对 `16/` 目录音频进行线性归一化，数值范围限定为 [-0.5, 0.5]          |
